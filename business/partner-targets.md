# Partner Targets

## Integration Roadmap

### 1. Google Cloud Platform (GCP) - BigQuery & AI Platform
**Free-tier Limits:**  
- BigQuery: 1TB/month of free queries  
- AI Platform: $300 credit/month  

**Integration Effort:** M  
**Value-add:**  
- **User Job:** Streamline data ingestion and preprocessing for large-scale time-series datasets.  
- **Rationale:** GCP's BigQuery offers robust data warehousing capabilities, ideal for handling the vast amounts of data required for accurate forecasting. Integrating with AI Platform allows users to leverage managed machine learning services, enhancing model training efficiency.  

**Revenue Share/Affiliate:** Yes, through GCP's Partner Advantage Program.

### 2. Microsoft Azure - Time Series Insights & Machine Learning Studio
**Free-tier Limits:**  
- Time Series Insights: 1GB/day storage, 100 events/sec  
- Machine Learning Studio: $200 credit/month  

**Integration Effort:** L  
**Value-add:**  
- **User Job:** Enhance real-time analytics and predictive maintenance capabilities.  
- **Rationale:** Azure's Time Series Insights provides advanced analytics tools for time-series data, while Machine Learning Studio offers a comprehensive platform for building and deploying ML models. This integration enables users to perform sophisticated forecasting tasks seamlessly.  

**Revenue Share/Affiliate:** Yes, via Azure's Co-Sell program.

### 3. AWS - SageMaker & QuickSight
**Free-tier Limits:**  
- SageMaker: 5 hours/month of ml.m4.xlarge instances  
- QuickSight: 300 dashboard views/month  

**Integration Effort:** M  
**Value-add:**  
- **User Job:** Simplify model deployment and visualization of forecasting results.  
- **Rationale:** AWS SageMaker facilitates easy deployment and management of ML models, while QuickSight provides powerful data visualization tools. Together, they offer a complete solution for end-to-end forecasting workflows.  

**Revenue Share/Affiliate:** Yes, through AWS's Partner Network.

### 4. Databricks - Unified Analytics Platform
**Free-tier Limits:** 1 cluster hour/day  
**Integration Effort:** L  
**Value-add:**  
- **User Job:** Accelerate data processing and model training using Spark.  
- **Rationale:** Databricks' platform integrates seamlessly with Apache Spark, providing a scalable environment for big data processing and ML model training. This integration significantly boosts the performance of forecast-loom for large datasets.  

**Revenue Share/Affiliate:** No direct affiliate, but potential for co-marketing opportunities.

### 5. Tableau - Data Visualization Software
**Free-tier Limits:** Tableau Public (limited features)  
**Integration Effort:** S  
**Value-add:**  
- **User Job:** Improve data exploration and result interpretation.  
- **Rationale:** Tableau is renowned for its intuitive data visualization capabilities. Integrating forecast-loom with Tableau enables users to create insightful visual representations of their forecasting results, aiding in better decision-making.  

**Revenue Share/Affiliate:** No direct affiliate, but potential for cross-promotion.

### 6. Grafana - Open-source Analytics & Monitoring Solution
**Free-tier Limits:** Unlimited (open-source)  
**Integration Effort:** S  
**Value-add:**  
- **User Job:** Monitor and visualize time-series data in real-time.  
- **Rationale:** Grafana's flexibility and extensive plugin ecosystem make it an excellent choice for real-time monitoring and visualization of time-series data. This integration enhances the operational visibility provided by forecast-loom.  

**Revenue Share/Affiliate:** No direct affiliate, but potential for community-driven support.

### 7. Segment - Customer Data Platform
**Free-tier Limits:** 10K events/month  
**Integration Effort:** M  
**Value-add:**  
- **User Job:** Unify customer data for personalized forecasting.  
- **Rationale:** Segment's CDP capabilities allow for the collection and unification of customer data from various sources. Integrating this with forecast-loom enables more accurate and personalized forecasting based on customer behavior.  

**Revenue Share/Affiliate:** No direct affiliate, but potential for strategic partnerships.

### 8. Matomo - Web Analytics Platform
**Free-tier Limits:** Unlimited (self-hosted)  
**Integration Effort:** S  
**Value-add:**  
- **User Job:** Analyze web traffic patterns for forecasting.  
- **Rationale:** Matomo provides detailed insights into web traffic and user behavior. Integrating this data with forecast-loom can enhance the accuracy of web-related time-series forecasts, such as website traffic predictions.  

**Revenue Share/Affiliate:** No direct affiliate, but potential for community-driven initiatives.
